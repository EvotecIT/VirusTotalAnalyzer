# Domain Monitoring List
# Format: domain.com[,additional-recipient@email.com,...]
#
# Simple format (sends alerts to default recipients only):
#   google.com
#
# With additional recipients (sends to default + specified emails):
#   critical-domain.com,ceo@company.com,security-team@company.com
#
# Lines starting with # are ignored
# Empty lines are ignored
